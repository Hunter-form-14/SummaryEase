<h1>SummaryEase</h1>

<h3>開発目的</h3>
リモート会議の効率化を目指し、テキストチャットから議事録を自動生成するシステムの開発を志しました。従来の議事録作成に要する時間と労力を削減しつつ、ユーザが任意に設定できる抽出項目により、各会議の目的に応じた柔軟な議事録生成を実現したいと考え、本システムを制作しました。

<h3>開発期間</h3>
学部3年次(2024年、ウェブプログラミング実習)

<h3>開発環境</h3>
使用言語:PHP、JavaScript
データベース:SQLite
API:ChatGPT-4
ハードウェア:Windows
その他:Miro(画面遷移図作成)、Lucid(ER図作成)

<h3>制作物の概要</h3>
4人チームで開発したリアルタイム議事録生成システムです。本システムでは、ユーザがテキストチャットで会議を行いながら、ChatGPT-4を活用して任意の抽出項目に基づいた議事録を自動生成できます。開発では、非同期処理とポーリング方式による通信機能の実装を担当し、PHPとJavaScriptのAsync/Await構文を用いてリアルタイム性の高いチャットシステムを構築しました。制作過程では、ネットワークエラーやタイミング依存のバグへの対応、サーバ負荷とリアルタイム性のバランス調整に苦心しましたが、綿密なエラーハンドリング設計とテストケースの実行により解決しました。
